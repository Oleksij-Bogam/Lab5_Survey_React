# Lab5 — React Survey Application

## Опис проєкту

Цей застосунок реалізує просту **web-форму опитування користувача** з використанням бібліотеки React.

Користувач вводить відповіді на запитання у текстові поля, після чого натискає кнопку **“Зберегти відповіді у файл”**. Введені дані формуються у текстовий файл і завантажуються браузером на комп’ютер користувача.

Метою лабораторної роботи є ознайомлення з принципами створення web-застосунків за допомогою React та реалізація застосунку **“Форма опитування”**.

## Функціональність

Застосунок дозволяє:

- відображати форму опитування;
- показувати користувачу список питань;
- вводити відповіді у текстові поля;
- вводити розгорнутий коментар у багаторядкове поле;
- зберігати відповіді у стані React;
- перевіряти, чи форма не є повністю порожньою;
- блокувати кнопку збереження, якщо всі поля порожні;
- очищати введені відповіді;
- формувати текстовий файл із результатами;
- завантажувати файл `survey_answers.txt`;
- відображати повідомлення про успішне збереження.

## Технології

- JavaScript;
- React;
- Vite;
- HTML;
- CSS;
- Browser File API;
- Blob API.

## Приклад роботи

1. Користувач запускає web-застосунок.
2. У браузері відкривається форма опитування.
3. Користувач вводить відповіді у текстові поля.
4. Для останнього питання користувач може ввести розгорнутий коментар.
5. Після введення даних кнопка **“Зберегти відповіді у файл”** стає активною.
6. Користувач натискає кнопку збереження.
7. Застосунок формує текстовий файл із відповідями.
8. Браузер завантажує файл `survey_answers.txt`.
9. На сторінці з’являється повідомлення про успішне збереження.

## Результат

Програма створює текстовий файл:

```text
survey_answers.txt
```

Файл завантажується браузером на комп’ютер користувача.

У файл записуються:

- назва результатів опитування;
- дата та час проходження опитування;
- список питань;
- відповіді користувача;
- значення `Не вказано`, якщо певне поле не було заповнене.

Приклад структури файлу:

```text
Результати опитування
Дата проходження: 10.05.2026, 21:34:12
-----------------------------------

1. Ваше ім’я:
Відповідь: Олексій

2. Яка ваша улюблена мова програмування?
Відповідь: C++

3. Який тип застосунків вам цікавіше розробляти?
Відповідь: web-застосунки

4. Що для вас найважливіше в інтерфейсі користувача?
Відповідь: зручність і зрозумілість

5. Ваш коментар або побажання:
Відповідь: тестовий коментар
```

## Структура основних файлів

Основна логіка застосунку розміщена у файлі:

```text
src/App.jsx
```

У цьому файлі реалізовано:

- створення функціонального компонента `App`;
- збереження списку питань;
- створення стану для відповідей користувача;
- обробку введення тексту;
- очищення форми;
- перевірку порожньої форми;
- формування тексту для файлу;
- створення файлу через `Blob`;
- запуск завантаження файлу через браузер.

Оформлення інтерфейсу розміщене у файлі:

```text
src/App.css
```

У цьому файлі реалізовано:

- стилі сторінки;
- оформлення основної картки форми;
- стилі заголовків і тексту;
- стилі полів введення;
- стилі кнопок;
- повідомлення про успішне збереження;
- адаптивність для екранів меншої ширини.

## Основні елементи інтерфейсу

У застосунку використано такі елементи:

- `h1` — для відображення головного заголовка;
- `p` — для короткого опису та повідомлень;
- `label` — для підписів до питань;
- `input` — для введення коротких відповідей;
- `textarea` — для введення розгорнутого коментаря;
- `button` — для збереження відповідей і очищення форми;
- `main` — для основного блоку застосунку;
- `section` — для групування питань.

## Основні елементи React

У застосунку використано:

- `useState` — для збереження відповідей користувача;
- `map()` — для виведення списку питань;
- `onChange` — для обробки введення тексту;
- `onClick` — для обробки натискання кнопок;
- умовне відображення — для показу повідомлення про успішне збереження;
- атрибут `disabled` — для блокування кнопки, якщо форма порожня.

## Запуск проєкту

Для запуску проєкту потрібно встановити залежності:

```bash
npm install
```

Після цього запустити локальний сервер:

```bash
npm run dev
```

Застосунок відкривається у браузері за адресою:

```text
http://localhost:5173/
```
